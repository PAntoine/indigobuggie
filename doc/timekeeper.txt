2. Description                                  *time-keeper-description*

This is a port of the main featured of the timekeeper plugin. It simply tracks
the time spent in the editor (typing) for the current project. Either the one
that is selected by the user, or auto generated by the current branch.

This feature is a core feature of |indigo-buggie|.

------------------------------------------------------------------------------
3. Configuring Time Keeper                      *time-keeper-configuring*

The feature is enabled by adding 'TimeKeeperFeature' to the
|g:IB_enabled_features| list.

    auto_tracking   Bool    Set if time keeper is to start tracking your time
                            automatically.

The timekeeper menu is quite simple an just will show a list of projects and
the accumulated time for the tasks in the project. 


------------------------------------------------------------------------------
5. Time Keeper Menu Commands                     *time-keeper-menu-commands*

The following are the commands that are supported for timekeeper:

  <cr>       Select item.
  Selects the item from the menu. Open/closes lines in the menu.

  p          Add a project to timekeeper.
  Starts a prompt to allow the user to add a project.

  j          Add a job.
  Starts a prompt to allow the user to add a job to the current or selected
  project.
  
  a          Amend a note.
  Change/Amends the note attached to the job. This will cause a window to open
  with the note for the job in it.

  s          Stop/Start time tracking.
  This will start/stop tracking. This is only useful if you are not using the
  auto-tracking feature.
